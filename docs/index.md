# Rust Hardware Playground

Welcome to the documentation for the Rust Hardware Playground! 

This project consists of two main parts:
1. **Rust / WASM Core:** The hardware simulation logic written in Rust.
2. **Next.js Frontend:** A visual simulator and dashboard.

## Overview
A pipelined dot product array behaves similarly to a single column of a systolic array. It computes the dot product of vectors as they stream through processing elements.

```rust
// Example of the Rust simulation tick
let state = sim.tick_and_get_state(&[10.0, 20.0]);
```
